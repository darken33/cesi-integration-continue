= Sonarqube : Analyse

== Lancer une analyse sonar

Via maven

[source, bash]
---- 
mvn clean package jacoco:report dependency-check:check sonar:sonar \
  -Dsonar.projectKey=<SONAR-PROJECT-KEY> \
  -Dsonar.host.url=<SONAR-URL> \
  -Dsonar.login=<SONAR-PROJECT-LOGIN>
----
